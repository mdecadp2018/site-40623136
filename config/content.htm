每週心得

Week1

國立虎尾科技大學機械設計工程系計算機程式 - 一乙第八週第三部分, 說明如何利用 Eric6 IDE 建立所需的 calculator 計算機程式對話表單. 過程中主要學習如何啟動 Eric6, 如何建立 Eric6 專案, 如何切換 Sources tab 與 Form tab, 如何在進入 Qt Designer 後利用格點布局, 建立按鈕與 Line Edit 部件.

week2

學會管理自己位於 https://github.com/mdecadp2018/ 中的期中作業倉儲, 然後在 W9期中前, 每週都要拍攝操作實習影片, 將教材內容與使用心得放入個人的網站、簡報與網誌中.到github倉儲中的Setting將branch內定分支切換到gh-pages分支,並且git clone倉儲.

week3

將p37置入隨身碟,在cmd clone的目錄下輸入python wsgi.py開啟近端,修改完內容後,記得要用generate_pages 轉為靜態網頁後,才能git add .→ git   commit -m →git push上推.還有開始學習使用Onshape繪圖.

week4

練習版本回推與結合:建立一個資料夾,在cmd目錄下輸入 git init,並到SciTE的Options下的Open Global Options Files檢查137行,確認過後在資料夾下輸入my progame新建一個txt檔,然後git add . → git commit -m "A",之後在txt檔內輸入first function再次儲存git add . → git commit -m "B",再來在txt檔輸入second function→save,到cmd 輸入git add . → git commit -m "C",在cmd輸入git log就可以看到版次,之後輸入git checkout 版次前7碼 切到B版次,之後輸入git branch dev→ git log→ git branch,但並沒有checkout過來,所以輸入git checkout dev,再來就是git add .→ git commit -m "D"→ git log→ git log --oneline --graph--all,再來輸入git checkout master轉到master再來合併,輸入git merge dev,但會有衝突,所以要選擇其中一個,所以在txt改版後,到cmd中輸入git add . → git commit -m "E",再次輸入git log --oneline --graph--all,如果要回到B版本,輸入git checkout 前7碼就ok了.
